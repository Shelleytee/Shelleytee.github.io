Curley Braces vs Double Quotes

Captivating thoughts from someone who tries not to think.
30 January 2017
Parentheses ( )

Parentheses are used for a number of different reasons JavaScript. They are used within mathmatical equations to represent grouping or prioritising of calculations. And, they are used within coding itself.

Examples are:

  • Mathmatical equations such as total = 59 * ((3 + 9) - ((8 - 7) * 2))
  • Define variable within a Function 
    function getTotalOfSum(num1, num2) {
  add = num1 + num2;
  return("Total" + add);
}
  • Call a Function getTotalOfSum(2, 9)
Brackets [ ]

Brackets are all about arrays, or a list of items (values) that can be found in a single place (variable).

An example is:

  • A toilet bag 
    var myToiletBag = ["toothbrush", "toothpaste", "soap", ["purse", "coins", 3]];
Braces { }

Braces, curly braces or curly brackets, help JavaScript to understand the structure of the script and what is wanted to be DONE (what action to be taken). There is always an opening curly brace and a closing curly brace.

Examples are:

  • Define the start and end of a Function
    var functionName = function() {                   //start brace
  console.log("Hello World");
}                                                 //end brace
  • Separate code into blocks within a Function
    var functionName = function(choice1, choice2) {   //start brace
  if (choice1 === choice2) {                      //start brace
    return ("The result is a tie");
  }                                               //end brace
}                                                 //end brace
  • Define the start and end of Loops
    while (boolean expression) {                      //start brace
  statement
}                                                 //end brace
  • Define the start and end of Conditional Statements
    if (boolean expression) {                         //start brace
  statement
} else {                                          //end brace & start brace
  statement
}                                                 //end brace
  • Defining an Objects contents
    var myDog = {                                     //start brace
  name: "Napolean",
  breed: "Staffie X",
  age: "8",
  friends: ["Jaz", "Michele"]
}                                                 //end brace
Single Quotes ' '

Indicate that whatever is between them is not code to be actioned, but simple text or values to be read and/or displayed. Their contents is better known as a string.

An example is:

  • Create a variable with the content of a sentence that does nothing except say "Hello World" var = 'Hello World'
Double Quotes " "

Does exactly the same as single quotes, but with double quotes. There is no difference between them for the program, it is more about which is more convenient for typing.

An example is:

  • Here us our variable with the content of the simple sentence var = "Hello World"
